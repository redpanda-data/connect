Pushes metrics using the https://github.com/statsd/statsd[StatsD protocol]. Supported tagging formats are 'none', 'datadog' and 'influxdb'.

```yml
# Config fields, showing default values

    flush_period: 100ms
    tag_format: none
  mapping: ""
```

*Type*: `string`

*Type*: `string`

*Default*: `"100ms"`
